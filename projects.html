Compress/Uncompress a File

C++

  • A C++ compression tool that uses Huffman encoding to compress and decompress text files.
  • The project consists of two main components:
    1. A compression program that creates encoded files with optimized character representation.
    2. A decompression program that reconstructs the original text.
  • The system uses a header file containing character frequency data (stored as 256 integers) to rebuild the Huffman tree during decompression.
  • Future plans include optimizing the header file format by storing the Huffman tree structure more efficiently using array indexing.
  • The project demonstrates practical applications of data compression algorithms and has sparked interest in encryption and data security.
Repo is private, but can be shared upon request.

Graph Project

C++

  • A C++ implementation of a custom graph class developed for an advanced data structures course at UCSD.
  • The project uses a map data structure to represent undirected graphs.
  • Includes several key algorithmic features:
    • Basic graph operations
    • Dijkstra's algorithm for weighted shortest paths
    • Modified BFS for unweighted shortest paths
    • Disjoint Set implementation for finding connected components with threshold constraints
  • The implementation focuses on runtime efficiency and practical usability while adhering to given API specifications.
Repo is private, but can be shared upon request.
